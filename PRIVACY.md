# Privacy Policy for Tab Memory Profiler

Last updated: December 20, 2025

## Data Collection
Tab Memory Profiler does NOT collect, transmit, or share any personal data.

## Data Storage
All data is stored locally on your device using Chrome's storage API:
- Tab access patterns (for AI predictions)
- Memory usage history
- User preferences

## Permissions Used
- **tabs**: To monitor tab memory usage
- **storage**: To save your settings locally
- **system.memory**: To measure memory consumption
- **alarms**: To periodically check memory

## Third Parties
We do not share data with any third parties.
