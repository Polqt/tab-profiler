# Tab Memory Profiler

Browser extension that monitors tab memory usage and suggests optimizations using AI.

## Features

- **Real-time Memory Monitoring** - Track memory usage for every open tab
- **Memory Leak Detection** - Automatically detect tabs consuming increasing memory
- **Smart AI Predictions** - ML model predicts which tabs you'll need next
- **Tab Hibernation** - One-click unload tabs to free memory (restored on click)
- **Domain Grouping** - See memory usage grouped by website
- **Visual Charts** - Interactive charts showing memory distribution
- **Bulk Actions** - Hibernate or close multiple idle tabs at once

## Tech Stack

- **WXT Framework** - Modern extension development framework
- **TypeScript** - Type-safe code
- **Vue 3** - Reactive UI components
- **Tailwind CSS** - Utility-first styling
- **Chart.js** - Data visualization
- **TensorFlow.js** - ML predictions
